이 23개의 패턴 크게 3가지로 나뉘어진다.

## 1. 생성 패턴
   1. ### Abstract Factory
   2. ### Builder
   3. ### Factory Method
   4. ### Prototype
   5. ### Singleon

## 2. 구조 패턴
   1. ### Adapter
   2. ### Bridge
   3. ### Composite
   4. ### Decorator
   5. ### Facade
   6. ### Flyweight
   7. ### Proxy

## 3. 행동 패턴

   2. ### Command
   3. ### Interpreter
   4. ### Iterator
   5. ### Mediator
   6. ### Memento
   7. ### Observer
   8. ### State
   9. ### Strategy
   10. ### Template Method
   11. ### Visitor
